# Pixetor

Pixetor is an easy-to-use sprite editor. It can be used to create game sprites, animations, pixel-art...

## Browser Support

Pixetor supports the following browsers:
* **Chrome** (latest)
* **Firefox** (latest)
* **Edge** (latest)
* **Internet Explorer** 11
